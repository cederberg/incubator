## General Guidelines

## Skill Writing
- Prefer open-ended instructions; agents are frequently competent to figure things out
- Only codify rules when evidence indicates a problem or to shortcut necessary exploration
- Batch edits to the same file in one pass, but first confirm everything with user
- After editing a skill, spawn a sub-agent reviewer to check brevity, clarity, usefulness, etc.
