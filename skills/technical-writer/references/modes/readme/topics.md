# Research Topics: README

Use these topics to guide your research scope. Each researcher will be assigned one topic from this list.

- **Purpose & audience** — what the project does, for whom, and in what context it is used; any notable design constraints or goals visible in the code or documentation
- **Setup & prerequisites** — what a user needs before they can use the project; how to install it; any runtime dependencies or platform requirements
- **Usage** — the primary commands, invocations, or API entry points; key options or parameters and their effects; representative input/output pairs
- **Configuration** — named configuration options, their defaults, and their effects on behaviour; where configuration lives (file, environment variable, flag)
- **Contribution & support** — how to build from source, how to run tests, how to contribute, where to get help or file issues
